1545. Find Kth Bit in Nth Binary String
Medium
Topics
Companies
Hint
Given two positive integers n and k, the binary string Sn is formed as follows:

S1 = "0"
Si = Si - 1 + "1" + reverse(invert(Si - 1)) for i > 1
Where + denotes the concatenation operation, reverse(x) returns the reversed string x, and invert(x) inverts all the bits in x (0 changes to 1 and 1 changes to 0).

For example, the first four strings in the above sequence are:

S1 = "0"
S2 = "011"
S3 = "0111001"
S4 = "011100110110001"
Return the kth bit in Sn. It is guaranteed that k is valid for the given n.

 

Example 1:

Input: n = 3, k = 1
Output: "0"
Explanation: S3 is "0111001".
The 1st bit is "0".
Example 2:

Input: n = 4, k = 11
Output: "1"
Explanation: S4 is "011100110110001".
The 11th bit is "1".
 

Constraints:

1 <= n <= 20
1 <= k <= 2n - 1


Solution 1:

class Solution:
    def findKthBit(self, n: int, k: int) -> str:
        def solve(length, k):
            # BASE CASE: S1 is always "0". 
            # When we've narrowed the search down to a length of 1, the bit must be "0".
            if length == 1:
                return "0"
            
            # The middle index of the current string Sn.
            # Since length is always (2^n - 1), it is always odd.
            mid = length // 2 + 1

            # CASE 1: k is exactly in the center.
            # According to the formula: Si = Si-1 + "1" + reverse(invert(Si-1))
            # The middle bit is always the "1" added during concatenation.
            if k == mid:
                return "1"
            
            # CASE 2: k is in the left half of the string.
            # The left half is identical to the previous string (Si-1).
            # We simply look for the kth bit in a string of half the length.
            elif k < mid:
                return solve(length // 2, k)
            
            # CASE 3: k is in the right half of the string.
            # The right half is Si-1 REVERSED and INVERTED.
            else:
                # 1. FIND MIRROR INDEX: 
                # To find the bit's original position in Si-1, we "un-reverse" k.
                # Formula: new_k = (total_length - current_k + 1)
                mirror_k = length - k + 1
                
                # 2. FIND AND INVERT:
                # We find the bit at that mirror position and flip it.
                # If solve() returns "0", we return "1". If "1", we return "0".
                res = solve(length // 2, mirror_k)
                return '1' if res == "0" else '0'

        # INITIALIZATION:
        # The length of Sn is always 2^n - 1. 
        # (1 << n) is a bitwise way to calculate 2 to the power of n.
        total_length = (1 << n) - 1
        
        # Start the recursive search
        return solve(total_length, k)
