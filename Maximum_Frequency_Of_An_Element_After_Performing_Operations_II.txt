You are given an integer array nums and two integers k and numOperations.

You must perform an operation numOperations times on nums, where in each operation you:

Select an index i that was not selected in any previous operations.
Add an integer in the range [-k, k] to nums[i].
Return the maximum possible frequency of any element in nums after performing the operations.

 

Example 1:

Input: nums = [1,4,5], k = 1, numOperations = 2

Output: 2

Explanation:

We can achieve a maximum frequency of two by:

Adding 0 to nums[1], after which nums becomes [1, 4, 5].
Adding -1 to nums[2], after which nums becomes [1, 4, 4].
Example 2:

Input: nums = [5,11,20,20], k = 5, numOperations = 1

Output: 2

Explanation:

We can achieve a maximum frequency of two by:

Adding 0 to nums[1].
 

Constraints:

1 <= nums.length <= 105
1 <= nums[i] <= 109
0 <= k <= 109
0 <= numOperations <= nums.length


Solution 1:

class Solution:
    def maxFrequency(self, nums: List[int], k: int, numOperations: int) -> int:
        # Maximum possible value we need to consider as a target
        max_target_value = max(nums)
        # Difference map (sweep line technique) to track how many numbers
        # can potentially be shifted to each target value
        diff_map = defaultdict(int)
        # Frequency map to count how many numbers are already equal to a value
        freq_map = defaultdict(int)

        # Step 1: Build the difference map and frequency map
        for num in nums:
            freq_map[num] += 1  # count occurrences of each number

            # Each number can be shifted into the range [num - k, num + k]
            left_bound = max(num - k, 0)
            right_bound = min(num + k, max_target_value)

            # Mark the contribution in the difference map
            diff_map[left_bound] += 1
            diff_map[right_bound + 1] -= 1

            # Ensure the number itself exists in diff_map (important for iteration)
            diff_map[num] += 0  

        # Step 2: Sweep through diff_map in sorted order to compute cumulative coverage
        result = 1
        cumulative_count = 0

        for target in sorted(diff_map.keys()):
            # Update cumulative count: how many numbers can be shifted to 'target'
            cumulative_count += diff_map[target]

            # Numbers already equal to 'target'
            already_equal = freq_map[target]

            # Numbers that could be shifted to 'target' but aren't equal yet
            need_conversion = cumulative_count - already_equal

            # We can only convert up to numOperations elements
            conversions_used = min(need_conversion, numOperations)

            # Best frequency achievable for this target
            result = max(result, already_equal + conversions_used)

        return result
