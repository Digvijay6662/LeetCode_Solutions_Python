2749. Minimum Operations to Make the Integer Zero
Medium
116
188
Companies
You are given two integers num1 and num2.

In one operation, you can choose integer i in the range [0, 60] and subtract 2i + num2 from num1.

Return the integer denoting the minimum number of operations needed to make num1 equal to 0.

If it is impossible to make num1 equal to 0, return -1.

 

Example 1:

Input: num1 = 3, num2 = -2
Output: 3
Explanation: We can make 3 equal to 0 with the following operations:
- We choose i = 2 and substract 22 + (-2) from 3, 3 - (4 + (-2)) = 1.
- We choose i = 2 and substract 22 + (-2) from 1, 1 - (4 + (-2)) = -1.
- We choose i = 0 and substract 20 + (-2) from -1, (-1) - (1 + (-2)) = 0.
It can be proven, that 3 is the minimum number of operations that we need to perform.
Example 2:

Input: num1 = 5, num2 = 7
Output: -1
Explanation: It can be proven, that it is impossible to make 5 equal to 0 with the given operation.
 

Constraints:

1 <= num1 <= 109
-109 <= num2 <= 109

Solution 1:

class Solution:
    def makeTheIntegerZero(self, num1: int, num2: int) -> int:
        # Try all possible operation counts from 1 to 60
        # (Why 60? Because 2^60 is already huge > 10^18, enough for constraints)
        for operations_count in range(1, 61):

            # Remaining value after subtracting num2 in each operation
            remaining_value = num1 - operations_count * num2

            # If remaining_value is negative, further operations will only make it smaller
            if remaining_value < 0:
                return -1

            # Check if remaining_value can be represented as the sum of exactly operations_count powers of 2
            # .bit_count() gives the minimum number of powers of 2 needed
            # Condition 1: bit_count <= operations_count → we can split powers into smaller ones if needed
            # Condition 2: operations_count <= remaining_value → can't have more operations than the value itself
            if remaining_value.bit_count() <= operations_count and operations_count <= remaining_value:
                return operations_count

        # If no valid operation count found
        return -1
