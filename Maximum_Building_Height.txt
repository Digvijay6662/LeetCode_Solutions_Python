You want to build n new buildings in a city. The new buildings will be built in a line and are labeled from 1 to n.

However, there are city restrictions on the heights of the new buildings:

The height of each building must be a non-negative integer.
The height of the first building must be 0.
The height difference between any two adjacent buildings cannot exceed 1.
Additionally, there are city restrictions on the maximum height of specific buildings. These restrictions are given as a 2D integer array restrictions where restrictions[i] = [idi, maxHeighti] indicates that building idi must have a height less than or equal to maxHeighti.

It is guaranteed that each building will appear at most once in restrictions, and building 1 will not be in restrictions.

Return the maximum possible height of the tallest building.

 

Example 1:


Input: n = 5, restrictions = [[2,1],[4,1]]
Output: 2
Explanation: The green area in the image indicates the maximum allowed height for each building.
We can build the buildings with heights [0,1,2,1,2], and the tallest building has a height of 2.
Example 2:


Input: n = 6, restrictions = []
Output: 5
Explanation: The green area in the image indicates the maximum allowed height for each building.
We can build the buildings with heights [0,1,2,3,4,5], and the tallest building has a height of 5.
Example 3:


Input: n = 10, restrictions = [[5,3],[2,5],[7,4],[10,3]]
Output: 5
Explanation: The green area in the image indicates the maximum allowed height for each building.
We can build the buildings with heights [0,1,2,3,3,4,4,5,4,3], and the tallest building has a height of 5.
 

Constraints:

2 <= n <= 109
0 <= restrictions.length <= min(n - 1, 105)
2 <= idi <= n
idi is unique.
0 <= maxHeighti <= 109


Solution 1:

class Solution:
    def maxBuilding(self, n: int, restrictions: List[List[int]]) -> int:
        # ── CORE INSIGHT ──
        # Between any two restricted buildings (anchors), heights must
        # rise and fall like a tent, constrained by the adjacency rule
        # (adjacent heights differ by at most 1).
        #
        # Given two anchors:
        #   left  at position p1 with max height h1
        #   right at position p2 with max height h2
        #
        # The tallest peak reachable between them:
        #   1. Gap   = p2 - p1            (positions apart)
        #   2. hDiff = |h2 - h1|         (height difference at anchors)
        #   3. Remaining slack = gap - hDiff  (steps left after compensating hDiff)
        #   4. Peak  = max(h1,h2) + (remaining_slack // 2)
        #              (split remaining slack equally on each side of the peak)

        # ── ADD SENTINEL ANCHORS ──
        # Building 1 is always fixed at height 0 (problem constraint).
        # Building n has no explicit restriction but is bounded by how
        # high we can reach from building 1: max height at position n
        # is at most n-1 (climbing one step per building from height 0).
        # Adding both as restrictions unifies all boundary cases.
        restrictions.append([1, 0])        # left boundary: building 1, height 0
        restrictions.append([n, n - 1])    # right boundary: building n, height n-1

        # Sort by building position so adjacent pairs are neighbours in the list
        restrictions.sort()

        # ── FORWARD PASS: propagate constraints LEFT → RIGHT ──
        # A building cannot be taller than its left neighbour's max height
        # plus the distance between them (since heights can rise by at most 1
        # per step, the highest you can reach from the left is h_left + dist).
        # We tighten each building's max height using this reachability from left.
        for i in range(1, len(restrictions)):
            position_gap       = restrictions[i][0] - restrictions[i - 1][0]
            max_from_left      = restrictions[i - 1][1] + position_gap
            # Tighten: building i can't exceed what's reachable from the left
            restrictions[i][1] = min(restrictions[i][1], max_from_left)

        # ── BACKWARD PASS: propagate constraints RIGHT → LEFT ──
        # Same logic in reverse: a building cannot be taller than its right
        # neighbour's max height plus the distance between them.
        # After the forward pass tightened right-side heights, we push those
        # tighter constraints back left so every anchor reflects BOTH directions.
        for i in range(len(restrictions) - 2, -1, -1):
            position_gap       = restrictions[i + 1][0] - restrictions[i][0]
            max_from_right     = restrictions[i + 1][1] + position_gap
            # Tighten: building i can't exceed what's reachable from the right
            restrictions[i][1] = min(restrictions[i][1], max_from_right)

        # ── FIND PEAK BETWEEN EACH ADJACENT PAIR OF ANCHORS ──
        # After both passes, every anchor has the tightest possible height cap.
        # Between each adjacent pair of anchors, the tallest reachable point
        # forms a "tent" shape — calculate its peak height using the formula.
        global_max_height = 0

        for i in range(1, len(restrictions)):
            left_pos,  left_height  = restrictions[i - 1]
            right_pos, right_height = restrictions[i]

            # Total positions between the two anchors (inclusive of both endpoints)
            gap = right_pos - left_pos

            # Height difference between the two anchor caps.
            # At minimum, heights must differ by |h2 - h1| just to connect them
            # (one side must climb or descend to bridge the height gap).
            height_diff = abs(right_height - left_height)

            # Remaining steps after bridging the height difference.
            # These leftover steps can be used to go UP and then come back DOWN,
            # adding (remaining // 2) extra height above the taller anchor.
            # Integer division because each up-step must be paired with a down-step.
            remaining_slack = gap - height_diff

            # Peak = start from the taller anchor, climb half the remaining slack
            peak_height = max(left_height, right_height) + remaining_slack // 2

            global_max_height = max(global_max_height, peak_height)

        return global_max_height
