Given an array intervals where intervals[i] = [li, ri] represent the interval [li, ri), remove all intervals that are covered by another interval in the list.

The interval [a, b) is covered by the interval [c, d) if and only if c <= a and b <= d.

Return the number of remaining intervals.

 

Example 1:

Input: intervals = [[1,4],[3,6],[2,8]]
Output: 2
Explanation: Interval [3,6] is covered by [2,8], therefore it is removed.
Example 2:

Input: intervals = [[1,4],[2,3]]
Output: 1
 

Constraints:

1 <= intervals.length <= 1000
intervals[i].length == 2
0 <= li < ri <= 105
All the given intervals are unique.


Solution 1:

class Solution:
    def removeCoveredIntervals(self, intervals: List[List[int]]) -> int:
        # Sort intervals:
        # 1. By starting point ascending
        # 2. If start is same, by ending point descending
        # This ensures that larger intervals come before smaller ones
        intervals.sort(key=lambda x: (x[0], -x[1]))

        # Track the farthest right endpoint we've seen so far
        max_end = 0
        # Count of intervals that remain after removing covered ones
        remaining = 0

        for start, end in intervals:
            # If current interval's end is greater than max_end,
            # it is not covered by any previous interval
            if end > max_end:
                remaining += 1
                max_end = end
            # Otherwise, it's covered (do nothing)

        return remaining
