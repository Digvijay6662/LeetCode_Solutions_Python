You are given a 2D matrix grid of size n x n. Initially, all cells of the grid are colored white. In one operation, you can select any cell of indices (i, j), and color black all the cells of the jth column starting from the top row down to the ith row.

The grid score is the sum of all grid[i][j] such that cell (i, j) is white and it has a horizontally adjacent black cell.

Return the maximum score that can be achieved after some number of operations.

 

Example 1:

Input: grid = [[0,0,0,0,0],[0,0,3,0,0],[0,1,0,0,0],[5,0,0,3,0],[0,0,0,0,2]]

Output: 11

Explanation:


In the first operation, we color all cells in column 1 down to row 3, and in the second operation, we color all cells in column 4 down to the last row. The score of the resulting grid is grid[3][0] + grid[1][2] + grid[3][3] which is equal to 11.

Example 2:

Input: grid = [[10,9,0,0,15],[7,1,0,8,0],[5,20,0,11,0],[0,0,0,1,2],[8,12,1,10,3]]

Output: 94

Explanation:


We perform operations on 1, 2, and 3 down to rows 1, 4, and 0, respectively. The score of the resulting grid is grid[0][0] + grid[1][0] + grid[2][1] + grid[4][1] + grid[1][3] + grid[2][3] + grid[3][3] + grid[4][3] + grid[0][4] which is equal to 94.

 

Constraints:

1 <= n == grid.length <= 100
n == grid[i].length
0 <= grid[i][j] <= 109


Solution 1:

class Solution:
    def maximumScore(self, grid: List[List[int]]) -> int:
        n = len(grid)

        # Build prefix sums for each column
        # prefix[row][col] = sum of grid[0..row-1][col-1]
        prefix = [[0] * (n + 1) for _ in range(n + 1)]
        for row in range(1, n + 1):
            for col in range(1, n + 1):
                prefix[row][col] = prefix[row - 1][col] + grid[row - 1][col - 1]
        
        @cache
        def solve(prevHeight: int, currCol: int, prevTaken: int) -> int:
            """
            prevHeight: cutoff row index in the previous column
            currCol: current column index we are processing
            prevTaken: 1 if previous column was painted, 0 otherwise
            """
            if currCol == n:  # Base case: all columns processed
                return 0

            bestScore = 0
            # Try all possible cutoff heights for current column
            for height in range(n + 1):
                prevColScore = 0
                currColScore = 0

                # Contribution from previous column adjacency
                if prevTaken == 0 and height > prevHeight:
                    prevColScore += prefix[height][currCol] - prefix[prevHeight][currCol]

                # Contribution from current column adjacency
                if prevHeight > height:
                    currColScore += prefix[prevHeight][currCol + 1] - prefix[height][currCol + 1]

                # Case 1: Paint current column up to 'height'
                scoreIfTaken = prevColScore + currColScore + solve(height, currCol + 1, 1)

                # Case 2: Do not paint current column
                scoreIfNotTaken = prevColScore + solve(height, currCol + 1, 0)

                # Take the better option
                bestScore = max(bestScore, scoreIfTaken, scoreIfNotTaken)
                
            return bestScore

        # Start from column 0, with no previous column painted and cutoff height = 0
        return solve(0, 0, 0)
