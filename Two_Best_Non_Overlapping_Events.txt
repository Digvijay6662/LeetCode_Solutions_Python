2054. Two Best Non-Overlapping Events
Solved
Medium

Hint
You are given a 0-indexed 2D integer array of events where events[i] = [startTimei, endTimei, valuei]. The ith event starts at startTimei and ends at endTimei, and if you attend this event, you will receive a value of valuei. You can choose at most two non-overlapping events to attend such that the sum of their values is maximized.

Return this maximum sum.

Note that the start time and end time is inclusive: that is, you cannot attend two events where one of them starts and the other ends at the same time. More specifically, if you attend an event with end time t, the next event must start at or after t + 1.

 

Example 1:


Input: events = [[1,3,2],[4,5,2],[2,4,3]]
Output: 4
Explanation: Choose the green events, 0 and 1 for a sum of 2 + 2 = 4.

Constraints:

2 <= events.length <= 105
events[i].length == 3
1 <= startTimei <= endTimei <= 109
1 <= valuei <= 106


Solution 1:

class Solution:
    def maxTwoEvents(self, events: List[List[int]]) -> int:
        # Number of events
        n = len(events)
        # Sort events by start time (important for binary search later)
        events.sort(key=lambda x: x[0])
        
        # Precompute suffix maximum values:
        # suffix_max[i] = maximum event value among events[i..n-1]
        suffix_max = [0] * n
        suffix_max[n - 1] = events[n - 1][2]
        for i in range(n - 2, -1, -1):
            suffix_max[i] = max(events[i][2], suffix_max[i + 1])

        # Build suffix max array from right to left
        for i in range(n - 2, -1, -1):
            # At each index, store the max of current event value and suffix_max of next index
            suffix_max[i] = max(events[i][2], suffix_max[i + 1])
        
        max_sum = 0  # track the maximum sum of values
        # Iterate over each event, treating it as the "first" event
        for i in range(n):
            curr_sum = events[i][2]  # value of current event
            # Binary search: find the first event whose startTime > events[i][1]
            # (since endTime is inclusive, next event must start strictly later)
            j = bisect_right(events, [events[i][1], float('inf'), float('inf')])
            # If such an event exists, add the best possible value from suffix_max
            if j < n:
                curr_sum += suffix_max[j]
            # Update maximum sum found so far
            max_sum = max(max_sum, curr_sum)
        
        return max_sum


Solution 2:

class Solution:
    def maxTwoEvents(self, events: List[List[int]]) -> int:
        timeline = []  # list to store event start and end markers
        # For each event, add two entries:
        # - [start, 1, value] → marker for event starting (candidate to attend)
        # - [end+1, 0, value] → marker for event ending (update max value seen so far)
        for start, end, value in events:
            timeline.append([start, 1, value])
            timeline.append([end + 1, 0, value])  # end+1 ensures non-overlap (inclusive end)

        # Sort timeline by time (then by state implicitly)
        timeline.sort()
        max_value_so_far = 0   # best single event value seen up to current time
        max_value_sum = 0      # best sum of two non-overlapping events

        # Sweep through the timeline
        for time, marker_type, value in timeline:
            if marker_type == 1:
                # Event starts: try combining this event with the best past event
                max_value_sum = max(max_value_sum, value + max_value_so_far)
            else:
                # Event ends: update the best single event value seen so far
                max_value_so_far = max(max_value_so_far, value)

        return max_value_sum
