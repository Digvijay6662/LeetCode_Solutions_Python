Given the root of a binary tree, split the binary tree into two subtrees by removing one edge such that the product of the sums of the subtrees is maximized.

Return the maximum product of the sums of the two subtrees. Since the answer may be too large, return it modulo 109 + 7.

Note that you need to maximize the answer before taking the mod and not after taking it.

 

Example 1:


Input: root = [1,2,3,4,5,6]
Output: 110
Explanation: Remove the red edge and get 2 binary trees with sum 11 and 10. Their product is 110 (11*10)
Example 2:


Input: root = [1,null,2,3,4,null,null,5,6]
Output: 90
Explanation: Remove the red edge and get 2 binary trees with sum 15 and 6.Their product is 90 (15*6)
 

Constraints:

The number of nodes in the tree is in the range [2, 5 * 104].
1 <= Node.val <= 104


Solution 1:

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def maxProduct(self, root: Optional[TreeNode]) -> int:
        MOD = 10**9 + 7   # modulus value as required by the problem
        best_product = 0  # variable to keep track of the maximum product found

        # -------------------------------
        # Step 1: Compute the total sum of all nodes in the tree
        # -------------------------------
        def compute_total_sum(node):
            """
            Recursively compute the sum of all nodes in the tree.
            Returns 0 if the node is None.
            """
            if not node:
                return 0
            return node.val + compute_total_sum(node.left) + compute_total_sum(node.right)

        total_sum = compute_total_sum(root)  # total sum of the entire tree

        # -------------------------------
        # Step 2: DFS to compute subtree sums and update max product
        # -------------------------------
        def compute_subtree_sum(node):
            """
            Recursively compute the sum of the subtree rooted at 'node'.
            While computing, also calculate the product of:
                subtree_sum * (total_sum - subtree_sum)
            which represents splitting the tree at the edge above this subtree.
            Update the global best_product if this product is larger.
            """
            nonlocal best_product
            if not node:
                return 0

            # Sum of current subtree = node value + left subtree sum + right subtree sum
            subtree_sum = node.val + compute_subtree_sum(node.left) + compute_subtree_sum(node.right)

            # We cannot split at the root (whole tree), so skip when subtree_sum == total_sum
            if subtree_sum != total_sum:
                # Product of the two parts after cutting this edge
                product = subtree_sum * (total_sum - subtree_sum)
                best_product = max(best_product, product)

            return subtree_sum

        # Run DFS to evaluate all possible cuts
        compute_subtree_sum(root)

        # Return the maximum product modulo 10^9 + 7
        return best_product % MOD
