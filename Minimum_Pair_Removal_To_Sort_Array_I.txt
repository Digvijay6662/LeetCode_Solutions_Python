Given an array nums, you can perform the following operation any number of times:

Select the adjacent pair with the minimum sum in nums. If multiple such pairs exist, choose the leftmost one.
Replace the pair with their sum.
Return the minimum number of operations needed to make the array non-decreasing.

An array is said to be non-decreasing if each element is greater than or equal to its previous element (if it exists).

 

Example 1:

Input: nums = [5,2,3,1]

Output: 2

Explanation:

The pair (3,1) has the minimum sum of 4. After replacement, nums = [5,2,4].
The pair (2,4) has the minimum sum of 6. After replacement, nums = [5,6].
The array nums became non-decreasing in two operations.

Example 2:

Input: nums = [1,2,2]

Output: 0

Explanation:

The array nums is already sorted.

 

Constraints:

1 <= nums.length <= 50
-1000 <= nums[i] <= 1000


Solution 1:

class Solution:
    def minimumPairRemoval(self, nums: List[int]) -> int:
        operations = 0   # Count how many merge operations we perform

        # Keep looping until the array becomes non-decreasing
        while True:
            is_sorted = True        # Flag to check if array is already non-decreasing
            min_pair_sum = float('inf')  # Track the minimum sum of adjacent pairs
            min_pair_index = -1     # Track the index of the leftmost minimum pair

            # Traverse through adjacent pairs
            for i in range(len(nums) - 1):
                # If any element is greater than the next, array is not sorted
                if nums[i] > nums[i + 1]:
                    is_sorted = False

                # Calculate sum of this adjacent pair
                pair_sum = nums[i] + nums[i + 1]

                # Update minimum pair sum and index if this pair is smaller
                if pair_sum < min_pair_sum:
                    min_pair_sum = pair_sum
                    min_pair_index = i

            # If array is already sorted, stop
            if is_sorted:
                break

            # Merge the chosen pair (replace with their sum)
            nums[min_pair_index] = min_pair_sum
            nums.pop(min_pair_index + 1)  # Remove the second element of the pair
            operations += 1               # Count this operation

        return operations
