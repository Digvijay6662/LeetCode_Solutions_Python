1590. Make Sum Divisible by P

Given an array of positive integers nums, remove the smallest subarray (possibly empty) such that the sum of the remaining elements is divisible by p. It is not allowed to remove the whole array.

Return the length of the smallest subarray that you need to remove, or -1 if it's impossible.

A subarray is defined as a contiguous block of elements in the array.

 

Example 1:

Input: nums = [3,1,4,2], p = 6
Output: 1
Explanation: The sum of the elements in nums is 10, which is not divisible by 6. We can remove the subarray [4], and the sum of the remaining elements is 6, which is divisible by 6.
Example 2:

Input: nums = [6,3,5,2], p = 9
Output: 2
Explanation: We cannot remove a single element to get a sum divisible by 9. The best way is to remove the subarray [5,2], leaving us with [6,3] with sum 9.
Example 3:

Input: nums = [1,2,3], p = 3
Output: 0
Explanation: Here the sum is 6. which is already divisible by 3. Thus we do not need to remove anything.
 

Constraints:

1 <= nums.length <= 105
1 <= nums[i] <= 109
1 <= p <= 109


Solution 1:

class Solution:
    def minSubarray(self, nums: List[int], p: int) -> int:
        n = len(nums)
        # Total sum modulo p → remainder we need to "fix"
        remainder_to_remove = sum(nums) % p
        # If already divisible, no need to remove anything
        if remainder_to_remove == 0:
            return 0
        # Dictionary to store prefix_sum % p → index
        # Initialize with {0: -1} to handle subarray starting at index 0
        prefix_mod_index = {0: -1}
        
        prefix_sum_mod = 0   # Running prefix sum modulo p
        min_length = n       # Result initialized to max possible (array length)
        
        for i in range(n):
            # Update prefix sum modulo p
            prefix_sum_mod = (prefix_sum_mod + nums[i]) % p
            # We want a previous prefix such that:
            # (prefix_sum_mod - remainder_to_remove) % p == previous_prefix
            needed_mod = (prefix_sum_mod - remainder_to_remove + p) % p
            # If such prefix exists, we can remove subarray between them
            if needed_mod in prefix_mod_index:
                min_length = min(min_length, i - prefix_mod_index[needed_mod])
            # Update dictionary with current prefix sum modulo p
            prefix_mod_index[prefix_sum_mod] = i
        
        # If min_length == n, it means removing whole array → not allowed
        return min_length if min_length < n else -1
