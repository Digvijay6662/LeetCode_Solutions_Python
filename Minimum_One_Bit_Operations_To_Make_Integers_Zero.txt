1611. Minimum One Bit Operations to Make Integers Zero

Hard

Hint
Given an integer n, you must transform it into 0 using the following operations any number of times:

Change the rightmost (0th) bit in the binary representation of n.
Change the ith bit in the binary representation of n if the (i-1)th bit is set to 1 and the (i-2)th through 0th bits are set to 0.
Return the minimum number of operations to transform n into 0.

 

Example 1:

Input: n = 3
Output: 2
Explanation: The binary representation of 3 is "11".
"11" -> "01" with the 2nd operation since the 0th bit is 1.
"01" -> "00" with the 1st operation.
Example 2:

Input: n = 6
Output: 4
Explanation: The binary representation of 6 is "110".
"110" -> "010" with the 2nd operation since the 1st bit is 1 and 0th through 0th bits are 0.
"010" -> "011" with the 1st operation.
"011" -> "001" with the 2nd operation since the 0th bit is 1.
"001" -> "000" with the 1st operation.
 

Constraints:

0 <= n <= 109


Solution 1:

class Solution:
    def minimumOneBitOperations(self, n: int) -> int:
        if n == 0:
            return 0

        dp = [0] * 30
        dp[0] = 1

        for i in range(1, 30):
            dp[i] = 2 * dp[i-1] + 1

        result = 0
        sign = 1

        for i in range(29, -1, -1):
            ith_bit = n & (1 << i)
            if ith_bit:
                result += dp[i] * sign
                sign *= -1

        return result



Solution 2:

class Solution:
    def minimumOneBitOperations(self, n: int) -> int:
        # Base case: if n is 0, no operations are required
        if n == 0:
            return 0
        
        # Step 1: Find the highest power of 2 (curr) that is <= n
        # k will be the index of this highest set bit
        k = 0
        curr = 1
        while (curr * 2) <= n:
            curr *= 2   # keep doubling until curr exceeds n
            k += 1      # track the bit position of the highest set bit
        
        # Step 2: Apply recursive formula
        # Formula: f(n) = (2^(k+1) - 1) - f(n ^ curr)
        #
        # Explanation:
        # - (2^(k+1) - 1) represents the maximum number of operations
        #   needed to clear all bits up to position k.
        # - n ^ curr flips the highest set bit, reducing the problem size.
        # - Subtracting f(n ^ curr) adjusts for the smaller subproblem.
        
        return 2 ** (k + 1) - 1 - self.minimumOneBitOperations(n ^ curr)
