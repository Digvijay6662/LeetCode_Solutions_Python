Given two numbers, hour and minutes, return the smaller angle (in degrees) formed between the hour and the minute hand.

Answers within 10-5 of the actual value will be accepted as correct.

 

Example 1:


Input: hour = 12, minutes = 30
Output: 165
Example 2:


Input: hour = 3, minutes = 30
Output: 75
Example 3:


Input: hour = 3, minutes = 15
Output: 7.5
 

Constraints:

1 <= hour <= 12
0 <= minutes <= 59


Solution 1:

class Solution:
    def angleClock(self, hour: int, minutes: int) -> float:
        # Normalize the hour to a 12-hour format.
        # Example: hour=12 should be treated as 0 (top of the clock).
        hour = hour % 12

        # Minute hand moves 360° in 60 minutes → 6° per minute.
        minute_angle = minutes * 6

        # Hour hand moves 360° in 12 hours → 30° per hour.
        # Additionally, it moves 0.5° per minute (since 30°/60 = 0.5).
        hour_angle = hour * 30 + minutes * 0.5

        # Find the absolute difference between the two angles.
        diff = abs(hour_angle - minute_angle)

        # The clock hands form two angles: diff and 360 - diff.
        # We want the smaller of the two.
        smaller_angle = min(diff, 360 - diff)

        # Return the smaller angle.
        return smaller_angle
