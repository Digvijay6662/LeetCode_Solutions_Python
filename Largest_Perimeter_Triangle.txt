Given an integer array nums, return the largest perimeter of a triangle with a non-zero area, formed from three of these lengths. If it is impossible to form any triangle of a non-zero area, return 0.

 

Example 1:

Input: nums = [2,1,2]
Output: 5
Explanation: You can form a triangle with three side lengths: 1, 2, and 2.
Example 2:

Input: nums = [1,2,1,10]
Output: 0
Explanation: 
You cannot use the side lengths 1, 1, and 2 to form a triangle.
You cannot use the side lengths 1, 1, and 10 to form a triangle.
You cannot use the side lengths 1, 2, and 10 to form a triangle.
As we cannot use any three side lengths to form a triangle of non-zero area, we return 0.
 

Constraints:

3 <= nums.length <= 104
1 <= nums[i] <= 106


Solution 1:

class Solution:
    def largestPerimeter(self, nums: List[int]) -> int:
        n = len(nums)              # Get the number of elements in the array
        nums.sort()                # Sort the array in non-decreasing order

        # Start from the largest element and move backwards
        # We check triplets (nums[i-2], nums[i-1], nums[i]) to see if they form a valid triangle
        for i in range(n - 1, 1, -1):
            # Triangle condition: sum of two smaller sides > largest side
            if nums[i] < nums[i - 1] + nums[i - 2]:
                # If valid, return the perimeter of this triangle
                return nums[i] + nums[i - 1] + nums[i - 2]

        # If no valid triangle can be formed, return 0
        return 0
