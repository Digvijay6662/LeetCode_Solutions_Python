You have n computers. You are given the integer n and a 0-indexed integer array batteries where the ith battery can run a computer for batteries[i] minutes. You are interested in running all n computers simultaneously using the given batteries.

Initially, you can insert at most one battery into each computer. After that and at any integer time moment, you can remove a battery from a computer and insert another battery any number of times. The inserted battery can be a totally new battery or a battery from another computer. You may assume that the removing and inserting processes take no time.

Note that the batteries cannot be recharged.

Return the maximum number of minutes you can run all the n computers simultaneously.

 

Example 1:


Input: n = 2, batteries = [3,3,3]
Output: 4
Explanation: 
Initially, insert battery 0 into the first computer and battery 1 into the second computer.
After two minutes, remove battery 1 from the second computer and insert battery 2 instead. Note that battery 1 can still run for one minute.
At the end of the third minute, battery 0 is drained, and you need to remove it from the first computer and insert battery 1 instead.
By the end of the fourth minute, battery 1 is also drained, and the first computer is no longer running.
We can run the two computers simultaneously for at most 4 minutes, so we return 4.

Example 2:


Input: n = 2, batteries = [1,1,1,1]
Output: 2
Explanation: 
Initially, insert battery 0 into the first computer and battery 2 into the second computer. 
After one minute, battery 0 and battery 2 are drained so you need to remove them and insert battery 1 into the first computer and battery 3 into the second computer. 
After another minute, battery 1 and battery 3 are also drained so the first and second computers are no longer running.
We can run the two computers simultaneously for at most 2 minutes, so we return 2.
 

Constraints:

1 <= n <= batteries.length <= 105
1 <= batteries[i] <= 109


Solution 1:

class Solution:
    def maxRunTime(self, n: int, batteries: List[int]) -> int:
        # Binary search range:
        # Minimum possible runtime = 1 minute
        # Maximum possible runtime = total energy / number of computers
        # (since each computer must run simultaneously)
        left, right = 1, sum(batteries) // n
        answer = -1   # store the best feasible runtime found

        def canRunFor(time: int) -> bool:
            """
            Check if it's possible to run all 'n' computers
            simultaneously for 'time' minutes.
            Each battery can contribute at most 'time' minutes,
            because one computer only needs 'time' minutes.
            """
            total_energy = 0
            for b in batteries:
                # Each battery contributes either its full capacity
                # or 'time' (whichever is smaller).
                total_energy += min(b, time)

            # To run 'n' computers for 'time' minutes,
            # we need at least n * time total energy.
            return total_energy >= n * time

        # Standard binary search for maximum feasible time
        while left <= right:
            mid = left + (right - left) // 2  # candidate runtime

            if canRunFor(mid):
                # If feasible, store result and try longer runtime
                answer = mid
                left = mid + 1
            else:
                # If not feasible, try shorter runtime
                right = mid - 1

        return answer

Solution 2:

class Solution:
    def maxRunTime(self, n: int, batteries: List[int]) -> int:
        # Step 1: Calculate total energy available across all batteries
        total_energy = sum(batteries)

        # Step 2: Sort batteries so we can easily access the largest one
        batteries.sort()

        # Step 3: Greedy adjustment
        # If the largest battery has more energy than the average share (total_energy // num_computers),
        # it means that battery cannot be fully utilized in a balanced distribution.
        # So we "assign" that battery to one computer and remove it from the pool.
        while batteries[-1] > total_energy // n:
            # Remove the largest battery from the pool
            largest_battery = batteries.pop()
            # Subtract its energy from the total pool
            total_energy -= largest_battery
            # Reduce the number of computers to balance among
            n -= 1
        # Step 4: After balancing, the maximum runtime is simply the average energy per computer
        return total_energy // n
