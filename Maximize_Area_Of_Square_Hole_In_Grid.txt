2943. Maximize Area of Square Hole in Grid
Solved
Medium
Companies
Hint
There is a grid with n + 2 horizontal bars and m + 2 vertical bars, and initially containing 1 x 1 unit cells.

The bars are 1-indexed.

You are given the two integers, n and m.

You are also given two integer arrays: hBars and vBars.

hBars contains distinct horizontal bars in the range [2, n + 1].
vBars contains distinct vertical bars in the range [2, m + 1].
You are allowed to remove bars that satisfy any of the following conditions:

If it is a horizontal bar, it must correspond to a value in hBars.
If it is a vertical bar, it must correspond to a value in vBars.
Return an integer denoting the maximum area of a square-shaped hole in the grid after removing some bars (possibly none).

Output: 4

The hole has an area of 4.
It can be shown that it is not possible to get a square hole with an area more than 4.
Hence, the answer is 4.

Constraints:

1 <= n <= 109
1 <= m <= 109
1 <= hBars.length <= 100
2 <= hBars[i] <= n + 1
1 <= vBars.length <= 100
2 <= vBars[i] <= m + 1
All values in hBars are distinct.
All values in vBars are distinct.


Solution 1:

class Solution:
    def maximizeSquareHoleArea(self, n: int, m: int, hBars: List[int], vBars: List[int]) -> int:
        hBars.sort()
        vBars.sort()

        lh = len(hBars)
        lv = len(vBars)
        max_ch = 0
        max_cv = 0

        i = 0
        while i < lh:
            ch = 1
            while i + 1 < lh and hBars[i] + 1 == hBars[i + 1]:
                ch += 1
                i += 1
            i += 1
            max_ch = max(max_ch, ch)

        
        i = 0
        while i < lv:
            cv = 1
            while i + 1 < lv and vBars[i] + 1 == vBars[i + 1]:
                cv += 1
                i += 1
            i += 1
            max_cv = max(max_cv, cv)

        return min(max_cv + 1, max_ch + 1) ** 2

class Solution:
    def maximizeSquareHoleArea(self, n: int, m: int, hBars: List[int], vBars: List[int]) -> int:
        # Sort the removable horizontal and vertical bars
        hBars.sort()
        vBars.sort()

        # Track current consecutive streaks and maximum streaks
        consecutive_h = consecutive_v = 1
        max_consecutive_h = max_consecutive_v = 1

        # Find the longest consecutive sequence in horizontal bars
        for i in range(1, len(hBars)):
            if hBars[i] == hBars[i - 1] + 1:   # If consecutive
                consecutive_h += 1
            else:
                consecutive_h = 1              # Reset streak
            max_consecutive_h = max(max_consecutive_h, consecutive_h)

        # Find the longest consecutive sequence in vertical bars
        for i in range(1, len(vBars)):
            if vBars[i] == vBars[i - 1] + 1:   # If consecutive
                consecutive_v += 1
            else:
                consecutive_v = 1              # Reset streak
            max_consecutive_v = max(max_consecutive_v, consecutive_v)

        # The maximum square hole side length is determined by the smaller of the two
        # (horizontal and vertical consecutive removals), plus 1 (since removing k consecutive bars
        # creates k+1 continuous gaps).
        max_side_length = min(max_consecutive_h + 1, max_consecutive_v + 1)

        # Return the area of the square hole
        return max_side_length ** 2
