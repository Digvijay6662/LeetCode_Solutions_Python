You are given an m x n matrix grid of positive integers. Your task is to determine if it is possible to make either one horizontal or one vertical cut on the grid such that:

Each of the two resulting sections formed by the cut is non-empty.
The sum of elements in both sections is equal, or can be made equal by discounting at most one single cell in total (from either section).
If a cell is discounted, the rest of the section must remain connected.
Return true if such a partition exists; otherwise, return false.

Note: A section is connected if every cell in it can be reached from any other cell by moving up, down, left, or right through other cells in the section.

 

Example 1:

Input: grid = [[1,4],[2,3]]

Output: true

Explanation:



A horizontal cut after the first row gives sums 1 + 4 = 5 and 2 + 3 = 5, which are equal. Thus, the answer is true.
Example 2:

Input: grid = [[1,2],[3,4]]

Output: true

Explanation:



A vertical cut after the first column gives sums 1 + 3 = 4 and 2 + 4 = 6.
By discounting 2 from the right section (6 - 2 = 4), both sections have equal sums and remain connected. Thus, the answer is true.
Example 3:

Input: grid = [[1,2,4],[2,3,5]]

Output: false

Explanation:



A horizontal cut after the first row gives 1 + 2 + 4 = 7 and 2 + 3 + 5 = 10.
By discounting 3 from the bottom section (10 - 3 = 7), both sections have equal sums, but they do not remain connected as it splits the bottom section into two parts ([2] and [5]). Thus, the answer is false.
Example 4:

Input: grid = [[4,1,8],[3,2,6]]

Output: false

Explanation:

No valid cut exists, so the answer is false.

 

Constraints:

1 <= m == grid.length <= 105
1 <= n == grid[i].length <= 105
2 <= m * n <= 105
1 <= grid[i][j] <= 105


Solution 1:

class Solution:
    def canPartitionGrid(self, grid: List[List[int]]) -> bool:

        def checkHorizontalCuts(matrix):
            """
            Check if a horizontal cut (splitting rows) can partition the matrix
            into two connected sections with equal sums, or sums that can be
            equalized by removing one valid cell.
            """
            nonlocal totalSum
            numRows, numCols = len(matrix), len(matrix[0])

            seenValues = set()   # Keep track of values encountered in the top section
            topSum = 0           # Sum of elements in the top section

            # Try cutting after each row (except the last one, since both sections must be non-empty)
            for rowIndex in range(numRows - 1):
                # Add current row’s values to the top section
                for colIndex in range(numCols):
                    seenValues.add(matrix[rowIndex][colIndex])
                    topSum += matrix[rowIndex][colIndex]

                # Bottom section sum = total - top
                bottomSum = totalSum - topSum
                diff = topSum - bottomSum  # Difference between top and bottom sums

                # Case 1: sums are already equal
                if diff == 0:
                    return True

                # Case 2: difference matches a boundary cell (removing it keeps connectivity)
                if diff == matrix[0][0]:          # top-left corner
                    return True
                if diff == matrix[0][numCols - 1]: # top-right corner
                    return True
                if diff == matrix[rowIndex][0]:   # leftmost cell of the cut row
                    return True

                # Case 3: difference matches any seen value (removing it keeps connectivity if not isolated)
                if rowIndex > 0 and numCols > 1 and diff in seenValues:
                    return True

            return False

        # Dimensions of the grid
        numRows, numCols = len(grid), len(grid[0])

        # Compute total sum of all cells
        totalSum = sum(sum(row) for row in grid)

        # --- Horizontal cuts ---
        if checkHorizontalCuts(grid):
            return True

        # Reverse rows (simulate cutting from bottom side)
        grid.reverse()
        if checkHorizontalCuts(grid):
            return True
        grid.reverse()  # Restore original grid

        # --- Vertical cuts ---
        # Transpose the grid to reuse horizontal cut logic for vertical cuts
        transposeGrid = [[grid[row][col] for row in range(numRows)] for col in range(numCols)]

        if checkHorizontalCuts(transposeGrid):
            return True

        # Reverse rows of transposed grid (simulate cutting from right side)
        transposeGrid.reverse()
        if checkHorizontalCuts(transposeGrid):
            return True

        # If no valid cut found
        return False
