You are given a positive integer n, representing an n x n city. You are also given a 2D grid buildings, where buildings[i] = [x, y] denotes a unique building located at coordinates [x, y].

A building is covered if there is at least one building in all four directions: left, right, above, and below.

Return the number of covered buildings.

 

Example 1:



Input: n = 3, buildings = [[1,2],[2,2],[3,2],[2,1],[2,3]]

Output: 1

Explanation:

Only building [2,2] is covered as it has at least one building:
above ([1,2])
below ([3,2])
left ([2,1])
right ([2,3])
Thus, the count of covered buildings is 1.
Example 2:



Input: n = 3, buildings = [[1,1],[1,2],[2,1],[2,2]]

Output: 0

Explanation:

No building has at least one building in all four directions.
Example 3:



Input: n = 5, buildings = [[1,3],[3,2],[3,3],[3,5],[5,3]]

Output: 1

Explanation:

Only building [3,3] is covered as it has at least one building:
above ([1,3])
below ([5,3])
left ([3,2])
right ([3,5])
Thus, the count of covered buildings is 1.
 

Constraints:

2 <= n <= 105
1 <= buildings.length <= 105 
buildings[i] = [x, y]
1 <= x, y <= n
All coordinates of buildings are unique.


Solution 1:

class Solution:
    def countCoveredBuildings(self, n: int, buildings: List[List[int]]) -> int:
        # For each column (x), track the minimum and maximum y values (vertical span)
        col_min_y, col_max_y = [float('inf')] * (n + 1), [float('-inf')] * (n + 1)
        # For each row (y), track the minimum and maximum x values (horizontal span)
        row_min_x, row_max_x = [float('inf')] * (n + 1), [float('-inf')] * (n + 1)
        # Step 1: Precompute min/max for each row and column
        for x, y in buildings:
            # Update vertical span for column x
            col_min_y[x] = min(col_min_y[x], y)
            col_max_y[x] = max(col_max_y[x], y)

            # Update horizontal span for row y
            row_min_x[y] = min(row_min_x[y], x)
            row_max_x[y] = max(row_max_x[y], x)

        covered_count = 0

        # Step 2: Check each building for coverage
        for x, y in buildings:
            # A building is covered if:
            # - There exists a building above (smaller y in same column)
            # - There exists a building below (larger y in same column)
            # - There exists a building to the left (smaller x in same row)
            # - There exists a building to the right (larger x in same row)
            if col_min_y[x] < y < col_max_y[x] and row_min_x[y] < x < row_max_x[y]:
                covered_count += 1

        return covered_count
