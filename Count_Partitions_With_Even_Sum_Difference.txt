Hint
You are given an integer array nums of length n.

A partition is defined as an index i where 0 <= i < n - 1, splitting the array into two non-empty subarrays such that:

Left subarray contains indices [0, i].
Right subarray contains indices [i + 1, n - 1].
Return the number of partitions where the difference between the sum of the left and right subarrays is even.

 

Example 1:

Input: nums = [10,10,3,7,6]

Output: 4

Explanation:

The 4 partitions are:

[10], [10, 3, 7, 6] with a sum difference of 10 - 26 = -16, which is even.
[10, 10], [3, 7, 6] with a sum difference of 20 - 16 = 4, which is even.
[10, 10, 3], [7, 6] with a sum difference of 23 - 13 = 10, which is even.
[10, 10, 3, 7], [6] with a sum difference of 30 - 6 = 24, which is even.
Example 2:

Input: nums = [1,2,2]

Output: 0

Explanation:

No partition results in an even sum difference.

Example 3:

Input: nums = [2,4,6,8]

Output: 3

Explanation:

All partitions result in an even sum difference.

 

Constraints:

2 <= n == nums.length <= 100
1 <= nums[i] <= 100


Solution 1:

class Solution:
    def countPartitions(self, nums: List[int]) -> int:
        n = len(nums)
        # left_sum will keep track of the sum of elements in the left partition
        # right_sum starts as the total sum of the array, and will shrink as we move elements to the left
        left_sum, right_sum = 0, sum(nums)
        # counter for valid partitions
        valid_partitions = 0
        # iterate through possible partition indices (0 to n-2)
        for i in range(n - 1):
            # move nums[i] from right partition to left partition
            left_sum += nums[i]
            right_sum -= nums[i]
            # difference between left and right sums
            diff = abs(left_sum - right_sum)
            # check if difference is even
            if diff % 2 == 0:
                valid_partitions += 1

        return valid_partitions


Solution 2:

class Solution:
    def countPartitions(self, nums: List[int]) -> int:
        # Step 1: Compute the total sum of the array
        total_sum = sum(nums)
        # Step 2: Get the length of the array
        n = len(nums)
        # Step 3: Key mathematical observation:
        # The difference between left and right sums at any partition is:
        #   diff = 2 * left_sum - total_sum
        #
        # Since 2 * left_sum is always EVEN:
        #   - If total_sum is EVEN → diff is always EVEN (because even - even = even).
        #   - If total_sum is ODD  → diff is always ODD  (because even - odd = odd).
        #
        # Therefore:
        #   - If total_sum is EVEN → every partition is valid.
        #   - If total_sum is ODD  → no partition is valid.
        
        # Step 4: If total sum is even, return (n - 1) because there are (n - 1) possible partitions
        if total_sum % 2 == 0:
            return n - 1
        else:
            # Step 5: If total sum is odd, return 0 because no partition can have even difference
            return 0
