2657. Find the Prefix Common Array of Two Arrays
Solved
Medium

You are given two 0-indexed integer permutations A and B of length n.

A prefix common array of A and B is an array C such that C[i] is equal to the count of numbers that are present at or before the index i in both A and B.

Return the prefix common array of A and B.

A sequence of n integers is called a permutation if it contains all integers from 1 to n exactly once.

 

Example 1:

Input: A = [1,3,2,4], B = [3,1,2,4]
Output: [0,2,3,4]
Explanation: At i = 0: no number is common, so C[0] = 0.
At i = 1: 1 and 3 are common in A and B, so C[1] = 2.
At i = 2: 1, 2, and 3 are common in A and B, so C[2] = 3.
At i = 3: 1, 2, 3, and 4 are common in A and B, so C[3] = 4.
Example 2:

Input: A = [2,3,1], B = [3,1,2]
Output: [0,1,3]
Explanation: At i = 0: no number is common, so C[0] = 0.
At i = 1: only 3 is common in A and B, so C[1] = 1.
At i = 2: 1, 2, and 3 are common in A and B, so C[2] = 3.
 

Constraints:

1 <= A.length == B.length == n <= 50
1 <= A[i], B[i] <= n
It is guaranteed that A and B are both a permutation of n integers.


Solution 1:

class Solution:
    def findThePrefixCommonArray(self, A: List[int], B: List[int]) -> List[int]:
        n = len(A)
        freq = defaultdict(int)
        result = []
        count = 0

        for i in range(n):
            freq[A[i]] += 1
            if freq[A[i]] == 2:
                count += 1
            freq[B[i]] += 1
            if freq[B[i]] == 2:
                count += 1
            result.append(count)

        return result

class Solution:
    def findThePrefixCommonArray(self, A: List[int], B: List[int]) -> List[int]:
        n = len(A)                     # Length of the permutations
        seenA, seenB = set(), set()     # Track elements seen so far in A and B
        ans = []                        # Final result array
        common_count = 0                # Running count of common elements

        for i in range(n):
            # Add the current elements from A and B into their respective sets
            seenA.add(A[i])
            seenB.add(B[i])

            # If the current element from A has already appeared in B's prefix,
            # it means this number is now common to both prefixes
            if A[i] in seenB:
                common_count += 1

            # Similarly, if the current element from B has already appeared in A's prefix,
            # it is also a new common element. 
            # The extra condition (B[i] != A[i]) avoids double-counting when both are the same.
            if B[i] in seenA and B[i] != A[i]:
                common_count += 1

            # Append the current count of common elements to the result
            ans.append(common_count)

        return ans

class Solution:
    def findThePrefixCommonArray(self, A: List[int], B: List[int]) -> List[int]:
        n = len(A)              # Length of the permutations
        ans = []                # Result array to store prefix common counts
        common = 0              # Running count of common elements so far
        seen = set()            # Set to track all elements seen from both arrays

        for i in range(n):
            # Process A[i]:
            # If A[i] was already added earlier (from B’s side),
            # then it means A[i] is now common to both prefixes
            if A[i] in seen:
                common += 1
            else:
                # Otherwise, mark A[i] as seen
                seen.add(A[i])
            
            # Process B[i]:
            # If B[i] was already added earlier (from A’s side),
            # then it means B[i] is now common to both prefixes
            if B[i] in seen:
                common += 1
            else:
                # Otherwise, mark B[i] as seen
                seen.add(B[i])
            
            # Append the current count of common elements to the result
            ans.append(common)
        
        return ans
