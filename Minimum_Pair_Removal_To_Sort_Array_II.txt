Given an array nums, you can perform the following operation any number of times:

Select the adjacent pair with the minimum sum in nums. If multiple such pairs exist, choose the leftmost one.
Replace the pair with their sum.
Return the minimum number of operations needed to make the array non-decreasing.

An array is said to be non-decreasing if each element is greater than or equal to its previous element (if it exists).

 

Example 1:

Input: nums = [5,2,3,1]

Output: 2

Explanation:

The pair (3,1) has the minimum sum of 4. After replacement, nums = [5,2,4].
The pair (2,4) has the minimum sum of 6. After replacement, nums = [5,6].
The array nums became non-decreasing in two operations.

Example 2:

Input: nums = [1,2,2]

Output: 0

Explanation:

The array nums is already sorted.

 

Constraints:

1 <= nums.length <= 105
-109 <= nums[i] <= 109


Solution 1:

class Solution:
    def minimumPairRemoval(self, nums: List[int]) -> int:
        n = len(nums)
        # prev[i] = index of the previous alive element before i
        # next[i] = index of the next alive element after i
        prev = [-1] * n
        next = [n] * n

        # SortedList will store tuples (pair_sum, index_of_left_element)
        # Always keeps pairs sorted by sum, then by index
        pair_sums = SortedList()

        bad_pairs = 0       # Count of adjacent pairs where nums[i] > nums[i+1]
        operations = 0      # Number of merge operations performed

        # Initialize linked list pointers and pair sums
        for i in range(n):
            prev[i] = i - 1
            next[i] = i + 1
            if i + 1 < n:
                # If nums[i] > nums[i+1], it's a "bad pair" (violates non-decreasing order)
                if nums[i] > nums[i + 1]:
                    bad_pairs += 1
                # Add the adjacent pair sum into the sorted structure
                pair_sums.add((nums[i] + nums[i + 1], i))

        # Keep merging until no bad pairs remain
        while bad_pairs > 0:
            # Extract the smallest sum pair (leftmost if tie)
            pair_sum, left = pair_sums.pop(0)
            right = next[left]

            # Skip if this pair is stale (already merged earlier)
            if right >= n or prev[right] != left:
                continue

            left_neighbor = prev[left]
            right_neighbor = next[right]

            # If left > right, this bad pair is being fixed
            if nums[left] > nums[right]:
                bad_pairs -= 1

            # --- Adjust bad_pairs count with left neighbor ---
            if left_neighbor >= 0:
                # Case 1: left_neighbor > left but <= merged sum → one bad pair fixed
                if nums[left_neighbor] > nums[left] and nums[left_neighbor] <= nums[left] + nums[right]:
                    bad_pairs -= 1
                # Case 2: left_neighbor <= left but > merged sum → new bad pair created
                elif nums[left_neighbor] <= nums[left] and nums[left_neighbor] > nums[left] + nums[right]:
                    bad_pairs += 1

                # Update pair_sums for (left_neighbor, left)
                if next[left_neighbor] == left:
                    pair_sums.discard((nums[left_neighbor] + nums[left], left_neighbor))
                    pair_sums.add((nums[left_neighbor] + nums[left] + nums[right], left_neighbor))

            # --- Adjust bad_pairs count with right neighbor ---
            if right_neighbor < n:
                # Case 1: right_neighbor >= right but < merged sum → new bad pair created
                if nums[right_neighbor] >= nums[right] and nums[right_neighbor] < nums[left] + nums[right]:
                    bad_pairs += 1
                # Case 2: right_neighbor < right but >= merged sum → one bad pair fixed
                elif nums[right_neighbor] < nums[right] and nums[right_neighbor] >= nums[left] + nums[right]:
                    bad_pairs -= 1

                # Update pair_sums for (left, right_neighbor)
                if prev[right_neighbor] == right:
                    pair_sums.discard((nums[right] + nums[right_neighbor], right))
                    pair_sums.add((nums[left] + nums[right] + nums[right_neighbor], left))
                    prev[right_neighbor] = left

            # --- Perform the merge ---
            next[left] = right_neighbor
            if right_neighbor < n:
                prev[right_neighbor] = left
            nums[left] += nums[right]   # Replace pair with their sum

            operations += 1

        return operations
