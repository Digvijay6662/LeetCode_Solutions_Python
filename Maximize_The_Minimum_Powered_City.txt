You are given a 0-indexed integer array stations of length n, where stations[i] represents the number of power stations in the ith city.

Each power station can provide power to every city in a fixed range. In other words, if the range is denoted by r, then a power station at city i can provide power to all cities j such that |i - j| <= r and 0 <= i, j <= n - 1.

Note that |x| denotes absolute value. For example, |7 - 5| = 2 and |3 - 10| = 7.
The power of a city is the total number of power stations it is being provided power from.

The government has sanctioned building k more power stations, each of which can be built in any city, and have the same range as the pre-existing ones.

Given the two integers r and k, return the maximum possible minimum power of a city, if the additional power stations are built optimally.

Note that you can build the k power stations in multiple cities.

 

Example 1:

Input: stations = [1,2,4,5,0], r = 1, k = 2
Output: 5
Explanation: 
One of the optimal ways is to install both the power stations at city 1. 
So stations will become [1,4,4,5,0].
- City 0 is provided by 1 + 4 = 5 power stations.
- City 1 is provided by 1 + 4 + 4 = 9 power stations.
- City 2 is provided by 4 + 4 + 5 = 13 power stations.
- City 3 is provided by 5 + 4 = 9 power stations.
- City 4 is provided by 5 + 0 = 5 power stations.
So the minimum power of a city is 5.
Since it is not possible to obtain a larger power, we return 5.
Example 2:

Input: stations = [4,4,4,4], r = 0, k = 3
Output: 4
Explanation: 
It can be proved that we cannot make the minimum power of a city greater than 4.
 

Constraints:

n == stations.length
1 <= n <= 105
0 <= stations[i] <= 105
0 <= r <= n - 1
0 <= k <= 109


Solution 1:

class Solution:
    def maxPower(self, stations: List[int], r: int, k: int) -> int:
        n = len(stations)
        # Difference array to efficiently calculate power coverage
        # diff[i] will represent the change in coverage starting at city i
        diff = [0] * (n + 1)
        
        # Build the initial coverage using the range of each station
        for city in range(n):
            left_bound = max(0, city - r)          # leftmost city covered by station at 'city'
            right_bound = min(n, city + r + 1)     # rightmost city covered (exclusive index)
            diff[left_bound] += stations[city]     # add coverage at left bound
            diff[right_bound] -= stations[city]    # remove coverage after right bound
        
        # Function to check if we can achieve at least 'target' minimum power in all cities
        def canAchieve(target):
            temp_diff = diff[:]        # copy of difference array (mutable for simulation)
            remaining_stations = k     # number of new stations we can still place
            current_power = 0          # running sum of coverage
            
            for city in range(n):
                current_power += temp_diff[city]   # update current coverage at this city
                
                # If this city has less power than target, we need to add extra stations
                if current_power < target:
                    needed = target - current_power
                    
                    # If we don't have enough stations left, fail
                    if remaining_stations < needed:
                        return False
                    
                    # Place 'needed' stations optimally at this city
                    remaining_stations -= needed
                    current_power += needed
                    
                    # These new stations will stop contributing after (city + 2*r + 1)
                    cutoff = min(n, city + 2 * r + 1)
                    temp_diff[cutoff] -= needed
            
            return True
        
        # Binary search for the maximum possible minimum power
        low, high = min(stations), sum(stations) + k
        best = low
        
        while low <= high:
            mid = (low + high) // 2
            if canAchieve(mid):
                best = mid       # mid is achievable, try for higher
                low = mid + 1
            else:
                high = mid - 1   # mid not achievable, try lower
        
        return best
