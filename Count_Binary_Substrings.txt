Given a binary string s, return the number of non-empty substrings that have the same number of 0's and 1's, and all the 0's and all the 1's in these substrings are grouped consecutively.

Substrings that occur multiple times are counted the number of times they occur.

 

Example 1:

Input: s = "00110011"
Output: 6
Explanation: There are 6 substrings that have equal number of consecutive 1's and 0's: "0011", "01", "1100", "10", "0011", and "01".
Notice that some of these substrings repeat and are counted the number of times they occur.
Also, "00110011" is not a valid substring because all the 0's (and 1's) are not grouped together.
Example 2:

Input: s = "10101"
Output: 4
Explanation: There are 4 substrings: "10", "01", "10", "01" that have equal number of consecutive 1's and 0's.
 

Constraints:

1 <= s.length <= 105
s[i] is either '0' or '1'.


Solution 1:

class Solution:
    def countBinarySubstrings(self, s: str) -> int:
        n = len(s)
        # prev_group_count: length of the previous consecutive group of identical chars
        # curr_group_count: length of the current consecutive group of identical chars
        prev_group_count, curr_group_count = 0, 1  
        
        # total_substrings: final result count
        total_substrings = 0
        
        # Traverse the string starting from the second character
        for i in range(1, n):
            if s[i] == s[i - 1]:
                # If current char is same as previous, extend current group
                curr_group_count += 1
            else:
                # If current char is different, 
                # valid substrings are limited by the smaller of prev and curr group sizes
                total_substrings += min(prev_group_count, curr_group_count)
                
                # Move current group to previous, reset current group count
                prev_group_count = curr_group_count
                curr_group_count = 1
        
        # After loop ends, add substrings formed by the last group
        total_substrings += min(prev_group_count, curr_group_count)
        
        return total_substrings
