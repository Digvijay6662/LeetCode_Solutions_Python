You are given an integer array prices representing the daily price history of a stock, where prices[i] is the stock price on the ith day.

A smooth descent period of a stock consists of one or more contiguous days such that the price on each day is lower than the price on the preceding day by exactly 1. The first day of the period is exempted from this rule.

Return the number of smooth descent periods.

 

Example 1:

Input: prices = [3,2,1,4]
Output: 7
Explanation: There are 7 smooth descent periods:
[3], [2], [1], [4], [3,2], [2,1], and [3,2,1]
Note that a period with one day is a smooth descent period by the definition.
Example 2:

Input: prices = [8,6,7,7]
Output: 4
Explanation: There are 4 smooth descent periods: [8], [6], [7], and [7]
Note that [8,6] is not a smooth descent period as 8 - 6 ≠ 1.
Example 3:

Input: prices = [1]
Output: 1
Explanation: There is 1 smooth descent period: [1]
 

Constraints:

1 <= prices.length <= 105
1 <= prices[i] <= 105


Solution 1:

class Solution:
    def getDescentPeriods(self, prices: List[int]) -> int:
        n = len(prices)
        # Initialize:
        # total_periods → total count of smooth descent periods
        # current_streak → length of current descent streak ending at current day
        total_periods = 1   # first day always counts as a descent period
        current_streak = 1  # streak starts with length 1 (single day)

        # Iterate through prices starting from the second day
        for i in range(1, n):
            # Check if today's price is exactly 1 less than yesterday's
            if prices[i-1] - 1 == prices[i]:
                # Extend the current descent streak
                current_streak += 1
            else:
                # Reset streak since condition breaks
                current_streak = 1

            # Add current streak length to total periods
            # (every extension creates new valid sub-periods)
            total_periods += current_streak

        return total_periods
