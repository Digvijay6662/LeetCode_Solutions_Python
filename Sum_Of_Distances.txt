You are given a 0-indexed integer array nums. There exists an array arr of length nums.length, where arr[i] is the sum of |i - j| over all j such that nums[j] == nums[i] and j != i. If there is no such j, set arr[i] to be 0.

Return the array arr.

 

Example 1:

Input: nums = [1,3,1,1,2]
Output: [5,0,3,4,0]
Explanation: 
When i = 0, nums[0] == nums[2] and nums[0] == nums[3]. Therefore, arr[0] = |0 - 2| + |0 - 3| = 5. 
When i = 1, arr[1] = 0 because there is no other index with value 3.
When i = 2, nums[2] == nums[0] and nums[2] == nums[3]. Therefore, arr[2] = |2 - 0| + |2 - 3| = 3. 
When i = 3, nums[3] == nums[0] and nums[3] == nums[2]. Therefore, arr[3] = |3 - 0| + |3 - 2| = 4. 
When i = 4, arr[4] = 0 because there is no other index with value 2. 

Example 2:

Input: nums = [0,5,3]
Output: [0,0,0]
Explanation: Since each element in nums is distinct, arr[i] = 0 for all i.
 

Constraints:

1 <= nums.length <= 105
0 <= nums[i] <= 109
 

Note: This question is the same as 2121: Intervals Between Identical Elements.


Solution 1:

class Solution:
    def distance(self, nums: List[int]) -> List[int]:
        n = len(nums)

        # Maps to track frequency and sum of indices for each number
        freq_map = defaultdict(int)       # count of occurrences of each number
        index_sum_map = defaultdict(int)  # sum of indices where each number appears

        result = [0] * n  # final answer array

        # -------- LEFT TO RIGHT PASS --------
        # For each index, calculate contribution from previous occurrences of nums[i]
        for i in range(n):
            freq = freq_map[nums[i]]          # how many times nums[i] has appeared before
            prev_index_sum = index_sum_map[nums[i]]  # sum of indices of those occurrences

            # Contribution formula:
            # distance = freq * i - prev_index_sum
            # Example: if nums[i] appeared at indices [a, b], then contribution = |i-a| + |i-b|
            result[i] += freq * i - prev_index_sum

            # Update maps with current index
            freq_map[nums[i]] += 1
            index_sum_map[nums[i]] += i

        # Clear maps for reuse in reverse pass
        freq_map.clear()
        index_sum_map.clear()

        # -------- RIGHT TO LEFT PASS --------
        # For each index, calculate contribution from future occurrences of nums[i]
        for i in range(n - 1, -1, -1):
            freq = freq_map[nums[i]]          # how many times nums[i] has appeared after i
            future_index_sum = index_sum_map[nums[i]]  # sum of indices of those future occurrences

            # Contribution formula:
            # distance = future_index_sum - freq * i
            result[i] += future_index_sum - freq * i

            # Update maps with current index
            freq_map[nums[i]] += 1
            index_sum_map[nums[i]] += i

        return result
