Hint
Given an integer n, return any array containing n unique integers such that they add up to 0.

 

Example 1:

Input: n = 5
Output: [-7,-1,1,3,4]
Explanation: These arrays also are accepted [-5,-1,1,2,3] , [-3,-1,2,-2,4].
Example 2:

Input: n = 3
Output: [-1,0,1]
Example 3:

Input: n = 1
Output: [0]
 

Constraints:

1 <= n <= 1000


Solution 1:

class Solution:
    def sumZero(self, n: int) -> List[int]:
        numbers = []  # List to store the result

        # If n is odd, include 0 in the list
        # This ensures the sum can still be zero while keeping numbers unique
        if n % 2 == 1:
            numbers.append(0)

        # Add pairs of positive and negative integers
        # Example: for n = 5, we add (1, -1), (2, -2)
        # These pairs always sum to 0
        for value in range(1, n // 2 + 1):
            numbers.append(value)    # positive number
            numbers.append(-value)   # its negative counterpart

        return numbers
